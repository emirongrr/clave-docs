---
title: Introduction
description: "Clave SDK"
---

Clave SDK is a library that allows you to integrate Clave into your application. You are able to use the SDK to
easily onboard users to Clave, and prepare transactions, implementing your own signing mechanism. Using the SDK, you
can access the account abstraction features that we have implemented in client side, without any zkSync and Smart Contract
Development knowledge.


### Available Platforms

We are currently supporting Typescript,and other clients are not implemented yet.
---
title: Initialize
description: "Clave SDK - Initialization"
---

The initialization process is where you set the parameters for the SDK to work with.
This is where you set the network provider, alias, public address and your signer method.

You can simple call the `ClaveSDK` class with the parameters you want to use, or with empty parameters to be set in the future.

<CodeGroup>

```ts With valid parameters
import { ClaveSDK } from '@getclave/sdk';
import { Provider } from "zksync-web3"

/** With valid parameters */
const provider = new Provider("RPC_URL")

const claveSDK = new ClaveSDK({
    provider,
    alias: "USERNAME",
    publicAddress: "0x...000",
    messageSignerFunction: async (alias, message) => {
        // Sign the transaction with the alias and return the signature
        return "SIGNATURE"
    }
})
```

```ts With empty parameters
import { ClaveSDK } from '@getclave/sdk';
import { Provider } from "zksync-web3"


const claveSDK = new ClaveSDK()

// You can init when you have the parameters

/** With valid parameters */
const provider = new Provider("RPC_URL")

claveSDK.init({
    provider,
    alias: "USERNAME",
    publicAddress: "0x...000",
    messageSignerFunction: async (alias, message) => {
        // Sign the transaction with the alias and return the signature
        return "SIGNATURE"
    }
})
```

</CodeGroup>

## API

```ts
export type ClaveSDKInitParams = {
    /* Web3 Provider */
    provider: Provider;

    /* User's username */
    alias: string;

    /* User's 20 byte public address */
    publicAddress: string;

    /* Function that signs the messages */
    messageSignerFunction: (
        alias: string,
        transaction: string,
    ) => Promise<string>;

    logLevel?: ClaveSDKLogLevel;
};

```
