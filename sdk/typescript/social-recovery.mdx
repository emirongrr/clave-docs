The module allows you utilizing social recovery features of Clave accounts.

<Warning>Most of the asynchronous methods, which interact with smart contracts, expects the SDK to be [initialized](/sdk/typescript/initialize).</Warning>

## Methods

`genGuardians` - Get all the social guaridans of user

```ts
const guardians = await claveSDK.socialRecovery.genGuardians();
```

## API Reference

The social recovery method of the Clave SDK implements the `IClaveSDKRecovery` interface, also adding some additional methods.

```ts
import { type PopulatedTransaction } from '@getclave/core';

import { type ClaveSDKError } from '../ClaveSDKError';

export interface IClaveSDKRecovery {
    /**
     * @returns {Promise<string | ClaveSDKError>} Nonce for recovery module
     */
    genRecoveryNonce(): Promise<string | ClaveSDKError>;

    /**
     * @returns {Promise<string | ClaveSDKError>} Signature for recovery in RS format
     */
    genRecoverySignature(
        recoveringAddress: string,
        newOwner: string,
    ): Promise<string | ClaveSDKError>;

    /**
     * @returns {Promise<PopulatedTransaction | ClaveSDKError>} Get whether the module is initialized
     */
    genIsInited(): Promise<boolean | ClaveSDKError>;

    /**
     * @returns {Promise<PopulatedTransaction | ClaveSDKError>} Populated transaction for stopping recovery
     */
    stopRecovery(): Promise<PopulatedTransaction | ClaveSDKError>;

    /**
     * @returns {Promise<boolean | ClaveSDKError>} Get whether the user is recovering
     */
    genIsRecovering(): Promise<boolean | ClaveSDKError>;

    /**
     * @returns {Promise<PopulatedTransaction | ClaveSDKError>} Populated transaction for starting recovery
     */
    startRecovery(
        recoveringAddress: string,
        newOwner: string,
        signature: string,
    ): Promise<PopulatedTransaction | ClaveSDKError>;

    /**
     * @returns {Promise<PopulatedTransaction | ClaveSDKError>} Populated transaction for executing recovery
     */
    executeRecovery(
        recoveringAddress: string,
    ): Promise<PopulatedTransaction | ClaveSDKError>;
}
```
