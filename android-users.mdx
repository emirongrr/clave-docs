Choose which password manager you're using and follow the guide to create an account: 
### [Google Password Manager](./google-password-manager)
- If you don't know which password manager your device support, you can check google password manager. Almost all Android devices support google password manager as a default option.
### [Samsung Pass](./samsung-pass)
- Samsung devices have an exclusive password manager that only works on Samsung, named Samsung Pass. So, if you're using a Samsung device and don't know which password manager you're using; you can navigate to [Samsung Pass Guide.](samsung-pass)
