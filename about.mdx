### What is Clave?

Clave is non-custodial account abstraction wallet on [zkSync](https://zksync.io/).
We are building the Clave with native account abstraction kit of zkSync, also utilizing the power of Account Abstraction and the Hardware Elements
(e.g [Secure Enclave](https://support.apple.com/en-gb/guide/security/sec59b0b31ff/web), [Android Trustzone](https://source.android.com/docs/security/features/trusty) etc.)
to create hardware level security.

### Frequently Asked Questions

<AccordionGroup>
  <Accordion title="Is Clave secure?">
    Clave is an application that turns your everyday device into a hardware wallet. No software or malicious actors can export your private key.
  </Accordion>

  <Accordion title="How can I use Clave?">
    Clave is currently under development. You can reach the platform specific versions from [getclave.io/download](https://getclave.io/download).
  </Accordion>

  <Accordion title="What is Account Abstraction?">
    Account Abstraction is an improvement on Ethereum that enables smart contracts to be used as a wallet, bringing many different new features such as batch transactions, sponsored transactions, and paymaster.
  </Accordion>

  <Accordion title="What is Secure Enclave?">
    As Apple defines the Secure Enclave is a dedicated secure subsystem integrated into Apple systems on chip (SoCs). The Secure Enclave is isolated from the main processor to provide an extra layer of security and is designed to keep sensitive user data secure even when the Application Processor kernel becomes compromised.
  </Accordion>
</AccordionGroup>
